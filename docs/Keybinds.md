# Game keybinds

## Keyboard
| Key | Action |
|----|---|
| W | Throttle |
| S | Brake |
| A | Steer left |
| D | Steer right |
| X | Shift between reverse/drive gears | 
| R | Shoot (WIP) |

## Controller
| Key | Action |
|----|---|
| X | Shift between reverse/drive gears |
| RT | Throttle |
| LT | Brake |
| Left Joystick | Steer |
| B | Shoot (WIP) |

# Debug Keybinds

- `F1` - Scene debug menu / inspector
- `F2` - Render debug menu
- `F3` - Physics debug menu
- `F4` - Input debug menu
- `F5` - Asset debug menu
- `F10` - Reload vehicle parameters (from json files)
